# Книга «Промпт-инжиниринг для LLM»

## Глава 1

Промпт-инжиниринг в простейшей форме - это искусство составления запроса так, чтобы ответ на него содержал всю информацию, необходимую, для решения поставленной задачи.

Вы должны осознавать, что эта книга не посвящена тому, как скрупулезно подбирать слова для одного точного промта, чтобы решить вашу задачу.

Когда мы говорим "промпт-инжиниринг", мы подразумеваем создание целого приложения на базе LLM. LLM-приложения выступают в качестве посредника, который итеративно и с сохранением состояния преобразует запросы пользователя в понятный для LLM текст и затем превращает полученные от модели данные в информацию и действия, отвечающие на изначальный запрос.
