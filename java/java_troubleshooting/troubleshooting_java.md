# Troubleshooting Java

## Chapter 7
- **Sampling (выборка)** - методика, в которой профилировщик используется, чтобы определить какой код выполняет приложение в конкретный момент времени.

Профилировщик сообщает нам имена классов, методов и порядок их вызовов.

Java-приложение отпраляет SQL-запросы в СУБД через драйвер JDBC. Профилировщик может перехватить любой SQL-запрос, отправленный в СУБД через драйвер JDBC.

Т.е. профилировщик перехватывает любые запросы независимо от технологии, применяемой для реализации уровня персистентности приложения.

Во многих случаях анализ медленного запроса требует его выполнения в SQL-клиенте, чтобы выяснить, какая часть запроса создает затруднения для СУБД.
